The good news is that ZAP does have Out of Band testing support via the OAST add-on - this supports BOAST and Interactsh. Also make sure you have a look at the Out-of-band Application Security Testing with OWASP ZAP blog post.

Remote API Access

On Day 11 Mic also mentioned that ZAP was not practical for testing mobile apps because the ZAP API is restricted to localhost.

While that is true by default it is also easy to change, as per the FAQ: How can I connect to ZAP remotely?
